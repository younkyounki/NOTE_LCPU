# HPC 学习笔记合集

本项目记录了每日 HPC 学习内容，结构清晰，适用于 GitHub 展示。

## 📁 项目结构示例

```
HPC-Notes-Final/
└── 2025-05-01/
    ├── note.md
    ├── image.png
    ├── image-1.png
    ├── ...
    └── image-13.png
```

## ✅ 使用说明

- 所有图片与 Markdown 文件在同一目录下，确保在 GitHub 上图片能正常显示。
- 图片引用使用相对路径，例如：

```markdown
![示意图](./image-1.png)
```

- 上传整个 `HPC-Notes-Final/` 文件夹到 GitHub 即可。
